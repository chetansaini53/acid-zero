# Security Policy

## Reporting a vulnerability in this project

If you find a security issue **in Acid Zero itself** (not in a target network),
please report it privately:

- **Email:** chetansaini53@gmail.com
- Please do **not** open a public GitHub issue for security vulnerabilities.
- Include reproduction steps and the affected file/version. I aim to acknowledge within a
  reasonable time and will credit reporters who wish to be named.

## Scope

This is a research/education project for **authorized** wireless testing. See
[ETHICS.md](ETHICS.md) for authorized-use terms. Reports about using this tool to attack
third parties are out of scope and are not supported.

## Responsible use

By using this software you accept [ETHICS.md](ETHICS.md) and the [LICENSE](LICENSE).
Only operate it against systems you own or are explicitly authorized in writing to test.
